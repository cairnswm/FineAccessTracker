Bulk API Documentation

The bulk.php endpoint allows you to insert multiple events into the database in a single request. The request body should contain an array of objects, each with the following fields:

  • page: The page identifier (string).
  • id: The item identifier (string).

Endpoint

POST /php/api/bulk.php

Headers

  • Content-Type: application/json
  • apikey: Your API key (string).

Request Body

{
    "data": [
        {
            "page": "examplePage",
            "id": "exampleId"
        },
        {
            "page": "anotherPage",
            "id": "anotherId"
        },
        {
            "error": "exampleError"
        },
        {
            "page": "",
            "id": "",
            "error": "",
            "type": "site"
        }
    ]
}

Response

On success, the API will return:

{
    "status": "success"
}

Sample JavaScript Call

const apiUrl = '/php/api/bulk.php';
const apiKey = 'your-api-key';

const requestData = {
    data: [
        {
            page: 'examplePage',
            id: 'exampleId'
        },
        {
            page: 'anotherPage',
            id: 'anotherId'
        },
        {
            error: 'exampleError'
        },
        {
            page: '',
            id: '',
            error: '',
            type: 'site'
        }
    ]
};

fetch(apiUrl, {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'apikey': apiKey
    },
    body: JSON.stringify(requestData)
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
